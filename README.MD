# Track Search

Bu dizin "Alotech Firması" Full Stack Developer Ödevi için oluşturulmuştur.
## Açıklama:
- Ön yüzde Admin-Lte,Bootstrap,Fontawesome ve Jquery kullanılmıştır.
- Retrieval işlemlerde Django Rest Framework kullanılmıştır.
